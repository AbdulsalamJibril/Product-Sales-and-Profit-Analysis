# Product-Sales-and-Profit-Analysis

### Project overview

This project analyzes product sales and profitability using an interactive dashboard. The goal is to provide a clear view of revenue, cost, and profit performance across products, time periods, locations, sales teams, and sales channels.

The dashboard summarizes key financial metrics and highlights areas that contribute most to overall profit. By visualizing trends and performance differences, the project helps stakeholders quickly understand business performance and identify key profit drivers.
